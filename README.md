# 🌑 Dark Terminal

Kodun karanlık tarafına hoş geldiniz.

## 🎨 Özellikler

- **WebGL Shader Animation**: Güzel hareketli 3D arka plan
- **Developer Console**: JavaScript komut satırı
- **Dark Theme**: Neon yeşil temaya sahip hacker estetik
- **Typewriter Effect**: Harf harf yazılan gizemli mesajlar
- **Responsive Design**: Mobil ve desktop uyumlu

## 🚀 Başlarken

```bash
# Depoyu klonla
git clone https://github.com/Reyhantkk/text-console.git

# Dosyaları aç
cd text-console

# Tarayıcıda aç
open index.html
```

## 💻 Komutlar

Console'da şu komutları kullanabilirsin:

```javascript
// Dil seçim
setLang("python")      // 🐍 Python
setLang("javascript")  // ⚡ JavaScript
setLang("rust")        // 🦀 Rust
setLang("go")          // 🐹 Go

// Yardım
help()                 // Komutları göster
clear()                // Console'u temizle
```

- **WebGL 2.0**: 3D rendering
- **JavaScript ES6+**: Interactive console
- **CSS3**: Dark UI tasarımı
